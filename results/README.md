## Column conventions (estimation CSVs)

| Column | Description |
|--------|-------------|
| `model` | Estimation block label (e.g., `B  detrended TWFE core`) |
| `estimator` | Estimator type (`TWFE`, `FD`, `C&S-DR`) |
| `coef` | Point estimate (log scale) |
| `se` | Standard error, clustered at bureau level |
| `pvalue` | Two-sided p-value |
| `N` | Number of observations |
| `R2` | Within-R² (where applicable) |

For event-study CSVs:

| Column | Description |
|--------|-------------|
| `rel_month` | Event time relative to ruling (−198 to +25) |
| `coef` | Coefficient estimate |
| `se` | Clustered standard error |
| `ci_lo`, `ci_hi` | 95% confidence interval bounds |
| `kind` | `bin` (annual) or `monthly` |
